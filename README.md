- [x] 字体图标


##### 安装使用
```
方式1：（推荐）
step01:复制package.json
step02:npm install

方式2：
git clone  -b v2.0.0 https://github.com/YMC-GitHub/cssLib-dev-tool.git
......

//开发
npm run dev

//构建输出
npm run build

//构建分析
npm run study
```
